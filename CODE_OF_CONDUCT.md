# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, personal attacks, and
discriminatory behavior are not tolerated. We are here to build a rigorous, honest
benchmark together.

Report unacceptable behavior to **aditya@latimal.com**. Reports will be handled
confidentially.
